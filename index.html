Personal Projects

Game Development

In-Perpetuum

In-Perpetuum is an idle focused game with satisfying visuals and audio that invites the player to set back and watch their system make money. This is done using a graph of gears that align and mesh together while charging to be able to pull a symbol for a reward. As the network is made of gears, players are able to change sizes to impact the speed and torque of the systems. These values impact different elements of the game. To create the visuals, several shaders are used to create a Kaleidoscope effect on the gears and an escher like background. This enable the player to stay involved by pulling for spins mindlessly between changes to their gear graphs. Art by Carson Diebold

Computer Graphics

I have created several shaders using in godot's shader language to create interesting visuals from base images. Godot's shader language is very similar to GLSL and can be converted easily. These shaders apply different transformations to find eachs targets positions. As the math remains the same, these can also be easily converted to screen shaders taking in what ever is in the currenct background. An interesting quirk is the transformations must be done in reverse order as GLSL can only get the pixel information for a point, not set it. So we have to find what point would create the effect at the current position.

Kaleidoscope Shader

A kaleidoscope effect is produced by angling two mirrors towards each other. As the angle gets tighter, more reflections are produced. Depending on the items reflected and the angle, different geometeric patterns are produced. At large angles it is easy to see the item used and creates larger patterns. Tight angles produce fast, circular patterns. These are used in the center of the gears for In-Perpetuum and rotate as the gear is rotated.

Escher Shader

This shader is based on this video. We take some of the transformations M.C. Escher frequently used and apply it dynamically to images. This shader allows the user to select the region between two circles, apply a recursive logarithmic spiral, then a steorographic projection. Now as the point is rotated, we are able to have spirals go in and out of each other.

Slice Shader

A simple slice shader intended to use with the kaleidoscope shader. This takes an image and creates a slice transition effect. If the number of slices is equal to the kaleidoscope slices, it can be used to have an image appear cleanly on top of the kaleidoscope effect. This can be used as a tool to create radial loading bars and other effects.
